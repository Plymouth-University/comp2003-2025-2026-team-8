# COMP2003 AI Finance Decision Support System
### Overview
This project is an AI powered financial decision support system. The goal of the system is to help users analyse financial markets and recieve simple trading signals based on machine learning predictions.

The system processes market data and financial news, applies a machine learning model, and produces a recommendation for each asset. The recommendation is presented as a **BUY, SELL, or HOLD** signal along with a confidence value.

The application also visualises historical prices and predicted outcomes through a web interface.

This project was developed as part of a COMP2003 group project on **Artificial Intelligence in Finance**.

### Running the Project
Follow these steps to run the system on your machine.

Requirements:
	- Python 3 installed
	- Node.js installed

Install Python packages:
	- pip install pandas yfinance feedparser vaderSentiment scikit-learn

!!There are TWO ways of running the web application!!

### Method 1 - Quick Way

#### 1. Navigate to the Project Folder
Go into the final-system-code folder

#### 2. Run the .bat File
Right-Click and run as administrator:
run_app.bat

This should automatically bring up the web app.
To shut down server just go to the original .bat terminal and press any key.
If running without administrator it may block the app but don't worry the .bat file is completely safe.
If there are any security concerns you can run it yourself with method 2.

### Method 2 - If there are issues with dependencies (Open the project folder in IDE)

#### 1. Navigate to the Project Folder
`cd final-system-code`

#### 2. Run the Backend (Python)
Install required packages:
`pip install pandas yfinance feedparser vaderSentiment scikit-learn`

Run the backend script:
`python ML.py`

This will:
* Download market data
* Process data
* Train the model
* Generate predictions
* Create the marketData.json file

#### 3. Run the Frontend (React)
Navigate to frontend (if needed):
`cd final-system-code`

Install dependencies:
`npm install`

Run the application:
`npm run dev`

Open the provided link in your browser

### Using the System
Once running, the system allows you to:
* View stock price charts
* See predicted price movement
* Recieve BUY, SELL, or HOLD signals
* View confidence levels
* Select different stocks
* Simulate investments using virtual currency
